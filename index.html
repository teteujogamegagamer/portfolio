Projeto da Corrida de Anjos vs Demônios

Este projeto é um jogo de corrida desenvolvido com a biblioteca p5.js, no qual cinco personagens representados por emojis de anjos e demônios disputam para ver quem alcança primeiro a linha de chegada. Cada jogador controla um dos seres espirituais utilizando as teclas A, S, J, K e L do teclado. Ao pressionar a tecla correspondente, o personagem avança uma distância aleatória. A corrida segue até que um dos competidores cruze a linha de chegada, momento em que o nome do vencedor é exibido na tela e o jogo é pausado automaticamente com noLoop(). O fundo muda de cor conforme o foco da janela do jogo, criando um efeito visual interessante. É um projeto simples e divertido, ideal para treinar lógica de programação, interação com o teclado e uso de funções gráficas básicas.

Projeto do Quente e frio

Este projeto é um mini jogo interativo desenvolvido com p5.js. Um ponto é colocado aleatoriamente na tela, mas permanece invisível para o jogador. Em vez de vê-lo diretamente, o jogador move o mouse e observa um círculo preto, cujo tamanho varia conforme a distância até o ponto escondido: quanto mais perto estiver, menor será o círculo. Essa dinâmica cria uma espécie de "busca quente e fria". Quando o mouse chega a menos de 3 pixels do ponto oculto, a mensagem "Encontrou, seu bobão!" é exibida na tela, e a animação é interrompida com noLoop(). É um ótimo exemplo de como aplicar lógica, cálculo de distância e interatividade de forma criativa.
